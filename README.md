## 404 and Error

Page Not Found and Error handling middlewares.

## Install

Install all packages

```bash
  npm install
```

# Run
Run and enjoy:

```bash
  npm start
```

# Test

Send this request via terminal

```bash
  curl localhost:3000/
```

or: 
```bash
  curl localhost:3000/wiki
```

or:
```bash
  curl localhost:3000/error
```

or:
```
  curl localhost:3000/somethingWord
```
